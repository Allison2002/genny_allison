Project 6 | The Animation

A UI/UX Case Study in Motion

Client Project | UI/UX-Focused

Mario Race is a lighthearted animation project created as part of my front-end development learning journey. The goal was to bring two iconic characters—Mario and Luigi—to life through racing animations using HTML, CSS, and vanilla JavaScript.

While the concept is simple, this project focuses heavily on animation timing, layout alignment, and UI polish. From image prep in Photoshop to responsive behavior and movement syncing in JavaScript, every detail was crafted to deliver a smooth, enjoyable experience.

Mario Race helped sharpen my animation sequencing, event timing, and cross-device layout skills. I built it entirely from scratch without jQuery or animation libraries—just pure code and creative problem-solving.
