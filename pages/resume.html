January 2023 – Present

AI Content Creator · SEO Specialist · Web Performance Optimizer

I founded Allison Stock Inc. as a performance-focused web development and AI content consultancy. My mission is to build SEO-optimized, fast-loading websites powered by cutting-edge AI tools and human-centered design principles.

I specialize in end-to-end development of content-rich platforms using HTML, CSS, JavaScript, WordPress, and performance tuning best practices. Each site is engineered to achieve top Core Web Vitals scores and maximum visibility through structured data, semantic markup, and technical SEO audits.

My creative toolkit includes ChatGPT, MidJourney, Figma, Canva, Photoshop, and Uizard. These tools power my workflows in AI content creation, prompt engineering, UX/UI wireframing, schema generation, and visual branding.

In parallel with client work, I’ve launched multiple personal case studies and portfolio projects that showcase how AI, performance metrics, and SEO intersect to deliver both search engine success and user satisfaction.

AI-Powered Content & Prompt Engineering: Created AI-generated blog articles, product descriptions, metadata, and schema using ChatGPT, optimized via LLM prompt design and structured workflows. Integrated prompt loops for performance-grade SEO writing and technical content generation.

Web Development & Optimization: Built fast, responsive websites with a focus on WCAG accessibility, Core Web Vitals, and Lighthouse 100/100 scores. Applied image compression, lazy loading, custom preload strategies, and responsive breakpoints for peak performance.

SEO & Structured Data: Conducted technical SEO audits, created schema markup, implemented Open Graph/Twitter cards, and optimized layouts to reduce CLS and LCP for both desktop and mobile.

Visual Design & Branding: Designed logos, UI systems, and page layouts using Figma, MidJourney, and Photoshop. Created social share cards, custom icons, and UI components tailored to the brand story of each client.

Client Collaboration: Delivered freelance UI/UX and performance optimization work using WordPress and Elementor. Translated goals into wireframes and prototypes, integrating newsletter forms, dynamic layouts, and mobile-first strategy.
