AI Content Creator & SEO Specialist Portfolio Project

Case Study: Texas Remodelers

Role: WordPress Developer, AI Content Creator, SEO Strategist

GitHub Repository: View the Code

Overview

The Texas Remodelers project is a commercial-style WordPress site created to showcase my ability to build functional, visually polished websites using WordPress. It highlights my adaptability with content management systems and my strategic approach to SEO and performance, even when facing unexpected technical challenges. This project remains a living work-in-progress, allowing me to document the recovery, optimization, and enhancement process transparently.

Objectives

  • Rebuild a WordPress site originally developed in Local and redeploy it on Netlify
  • Demonstrate core competencies in WordPress theming, structure, and plugin configuration
  • Optimize content and layout for SEO and Core Web Vitals
  • Maintain a commercial design aesthetic aligned with remodeling and construction brands

Development Stack

  • Platform: WordPress (built in Local and migrated to Netlify)
  • Design Tools:Canva (graphics), MidJourney (visual concepts), Figma (early layout planning)
  • AI Tools: ChatGPT (UX writing, metadata)
  • SEO Tools: SEMrush, Google Search Console (ongoing)
  • Hosting: Netlify (static deployment of recovered WordPress files)

Challenges & Solutions

  • Challenge: The original site broke after terminal command-line changes made during a school OS migration to Ubuntu.
    • Solution: Recovered the WordPress build and redeployed it as a static site on Netlify.
  • Challenge: Unable to access full WordPress admin or dynamic functionality in the current state.
    • Solution: Began optimizing the static output while planning for restoration of full CMS capabilities.

The Role of AI

  • ChatGPT: Generated metadata, UX copy, and alt tags based on commercial remodeling intent and user search patterns.
  • MidJourney & Canva: Used to create and polish design mockups reflecting a modern remodeling brand.
  • Future Steps: Replace static text and imagery with dynamic content and SEO plugins like RankMath or Yoast upon WordPress restoration.

SEO & Performance Strategy

  • Initial Baseline: SEO and Core Web Vitals scores are being tracked from the "starting point" to the final 100/100 goal
  • PageSpeed Goals: Documenting improvements in image formats, preload strategies, and layout shift reduction
  • SEO Enhancements: Schema markup, semantic HTML structure, proper heading hierarchy, and accessible image alt text

Outcome (In Progress)

Although the current site is not yet SEO-perfect, it acts as a real-world case study showing how a developer can recover, adapt, and continuously optimize a WordPress project even under broken conditions. The project stands as a testament to:

  • Problem-solving under pressure
  • Cross-platform adaptability
  • Commitment to iterative SEO improvement
