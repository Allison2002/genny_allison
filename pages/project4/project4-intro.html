project 4 | the website

texas remodelers

AI Content Creator & SEO Specialist Portfolio Project

Texas Remodelers is a professional website concept crafted to showcase my ability to create commercial-style sites using WordPress and AI-enhanced tools. This project is a cornerstone in my portfolio, reflecting my skill in designing clean, responsive layouts optimized for both user experience and search engine performance.

As part of this project, I also created the Texas Remodelers Blog, a sister site housed under the About tab in the site’s navigation menu. This blog highlights AI-powered content writing strategies, sample posts, and community-focused updates—reinforcing my content creation skills and demonstrating blog integration within a WordPress-based commercial layout.

Originally developed in Local for WordPress, this project faced unexpected setbacks during a system-wide environment shift at my school. Despite the disruption, I successfully recovered and rehosted the site on Netlify, where it now serves as an example of my ability to adapt and restore web projects under challenging circumstances.

While the site is not yet fully optimized for SEO or Core Web Vitals, it is actively being refined. I’ve included early performance data and marked this as a work in progress to transparently reflect my iterative process and commitment to quality. The final goal: a 100/100 PageSpeed Insights score and full WCAG alignment.

This project integrates AI-generated content, Canva visuals, and manual WordPress editing, making it a strong representation of my hybrid development style. It highlights my range—from code to content, design to optimization—and serves as proof of my capabilities with WordPress, even when working outside traditional hosting environments.
