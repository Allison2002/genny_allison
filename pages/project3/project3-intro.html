project 3 | the website

sports tube community

AI Content Creator & SEO Specialist Portfolio Project

Sports Tube Community Blog is the sister site to the Sports Tube Video Blog, built as a modern, AI-optimized platform where sports fans can publish and share their own articles. This dynamic blog project highlights my expertise in AI-powered content creation, SEO-driven development, and user-focused digital design.

From concept to deployment, this project was created to demonstrate how AI-assisted storytelling and fast-loading web performance can coexist in a seamless user experience. With a responsive layout and custom CMS-style article structure, the site delivers both flexibility and performance. It proudly holds a 100/100 score on Google PageSpeed Insights across mobile and desktop, ensuring optimal delivery and Core Web Vitals compliance.

I used ChatGPT to develop copy and support UX writing, while MidJourney and Canva provided striking, AI-enhanced visuals. The structure and layout were carefully mapped using Figma and Relume.io, with added refinements using Uizard for mobile-first responsiveness and WCAG accessibility alignment.

Hosted on Netlify and supported by Cloudinary for optimized image delivery, Sports Tube Community is a living example of scalable, community-focused content built with performance, usability, and search discoverability at its core.
