Client Project | UI/UX-Focused WordPress Redesign

Parent Tech Support hired me through Upwork to redesign and polish their blog and video resource pages. My focus was improving layout clarity, responsiveness, and user interaction using WordPress and Elementor.

I worked within an existing WordPress installation, using Elementor, Classic Editor, and All-in-One Video Gallery to structure the content. I ensured video thumbnails remained visible after playback, added a newsletter signup section, and refined the overall visual hierarchy.

Because this was a contract job, I didn’t have access to the full source code—but I delivered all work within WordPress, with screenshots and live links showcasing the final design. This project represents my ability to collaborate within existing systems and polish a client’s vision with thoughtful UX improvements.

The client left a 5-star review and shared their satisfaction with how the layout improved usability. Scroll below to see the final implementations and live examples.
